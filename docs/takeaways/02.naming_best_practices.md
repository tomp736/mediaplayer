# Naming Best Practices

1. Use snake_case for variables, functions, and method names.
2. Use PascalCase for class names.
3. Start variables, functions, and methods with a lowercase letter.
4. Start class names with an uppercase letter.
5. Avoid using single-letter names unless used as counters/indexes.
6. Avoid using Python keywords and built-in names as variable names.
7. Keep names short and descriptive.
8. Use descriptive names for constants, use all uppercase letters with underscores.
